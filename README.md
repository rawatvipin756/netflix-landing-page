# 🎬 Netflix Landing Page Clone

A responsive Netflix-inspired landing page built using HTML and CSS. This project focuses on recreating the layout and design while practicing modern frontend development concepts.

## 🚀 Features

- Responsive Navigation Bar
- Hero Section with Background Image
- Email Subscription Form
- Trending Movies Section
- Reasons to Join
- FAQ Section
- Footer with Social Links
- Hover Animations
- Responsive Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- CSS Positioning
- CSS Transitions
